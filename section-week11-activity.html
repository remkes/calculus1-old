Section 11.4 Week 11 Activity

Subsection 11.4.1 Extrema

Activity 11.4.1.

Find the \(x\) values of the maxima and minima of this function.

\begin{equation*} f(x) = 7x^2 - 2x - 4 \end{equation*}
Solution

First we take the derivative.

\begin{equation*} \frac{df}{dx} = 14x - 2 \end{equation*}

Next we set the derivative to zero and solve.

\begin{align*} 14x - 2 \amp = 0 \\ x \amp = \frac{1}{7} \end{align*}

The domain of this function is all real numbers. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp\left(-\infty, \frac{1}{7} \right) \amp \amp\left( \frac{1}{7}, \infty \right)\\ \amp f^\prime (0) = -2 \amp \amp f^\prime (1) = 12 \\ \amp \text{Negative} \amp \amp \text{Positive} \\ \amp f \text{ is decreasing.} \amp \amp f \text{ is increasing.} \end{align*}

Since the function changes from decreasing to increasing at the critical point, we must have a minimum at \(x = \frac{1}{7}\text{.}\)

Activity 11.4.2.

Find the \(x\) values of the maxima and minima of this function.

\begin{equation*} f(x) = \frac{x-1}{x^2+7} \end{equation*}
Solution

First we take the derivative, using the quotient rule.

\begin{align*} \frac{df}{dx} \amp = \frac{(x^2+7) \left( \frac{d}{dx} x-1 \right) - (x-1) \left( \frac{d}{dx} x^2+7 \right)}{(x^2+7)^2} \\ \amp = \frac{(x^2+7) - (x-1) (2x)}{(x^2+7)^2} \\ \amp = \frac{x^2+7 - 2x^2 + 2x}{(x^2+7)^2} \\ \amp = \frac{-x^2 + 2x + 7}{(x^2+7)^2} \end{align*}

Next we set the derivative to zero and solve. We only need to set the numerator equal to zero.

\begin{align*} -x^2 + 2x + 7 \amp = 0 \\ x^2 - 2x - 7 \amp = 0 \\ x \amp = \frac{2 \pm \sqrt{ 4 - (4)(1)(-7)}}{2}\\ x \amp = \frac{2 \pm \sqrt{32}}{2} = \frac{ 2 \pm 4 \sqrt{8}}{2} = 1 \pm \sqrt{8} \end{align*}

The domain of this function is all real numbers. We split up this domain using the critical points to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left( -\infty, 1 - \sqrt{8} \right) \amp \amp \left( 1 - \sqrt{8}, 1 + \sqrt{8}, \right) \amp \amp \left( 1 + \sqrt{8}, \infty \right)\\ \amp f^\prime (-2) = \frac{-1}{121} \amp \amp f^\prime (0) = \frac{1}{7} \amp \amp f^\prime (4) = \frac{-1}{519} \\ \amp \text{Negative} \amp \amp \text{Positive} \amp \amp \text{Negative} \\ \amp f \text{ is decreasing.} \amp \amp f \text{ is increasing.} \amp \amp f \text{ is decreasing.} \end{align*}

At \(x = 1 - \sqrt{8}\text{,}\) the function changes from decreasing to inscreasing, so this point is a minimum. At \(x = 1 + \sqrt{8}\text{,}\) the function changes from increasing to decreasing, so this point is a maximum.

Activity 11.4.3.

Find the \(x\) values of the maxima and minima of this function.

\begin{equation*} f(x) = e^{x^2-4} \end{equation*}
Solution

First we take the derivative, using the chain rule (I haven't shown the steps here).

\begin{equation*} \frac{df}{dx} = (2x)e^{x^2-4} \end{equation*}

Next we set the derivative to zero and solve. Note that the exponential term can never be zero, so we are free to divide by it, leaving only the polynomial term remaining.

\begin{align*} 2xe^{x^2-4} \amp = 0 \implies 2x = 0 \\ x \amp = 0 \end{align*}

The domain of this function is all real numbers. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(-\infty, 0 \right) \amp \amp \left( 0 \infty \right)\\ \amp f^\prime (-1) = -2e^{-3} \amp \amp f^\prime (1) = 2e^{-3}\\ \amp \text{Negative} \amp \amp \text{Positive} \\ \amp f \text{ is decreasing.} \amp \amp f \text{ is increasing.} \end{align*}

Since the function change from decreasing to increasing at the critical point, we must have a minimum at \(x = 0\text{.}\)

Activity 11.4.4.

Find the \(x\) values of the maxima and minima of this function.

\begin{equation*} f(x) = \frac{1}{x^2 - 6x + 10} \end{equation*}
Solution

First we take the derivative. I used the chain rule (with \(u^{-1}\) as the outside function), but the quotient rule works just a well.

\begin{equation*} \frac{df}{dx} = \frac{-(2x-6)}{(x^2-6x+10)^2} \end{equation*}

Next we set the derivative to zero and solve. Note we only need to set the numerator equal to zero.

\begin{align*} 2x-6 \amp = 0 \\ x \amp = 3 \end{align*}

This is not a root of the denominator, so this is a valid critical point. The domain of this function is all real numbers (since the denominator has no roots). We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(-\infty, 3 \right) \amp \amp \left( 3, \infty \right)\\ \amp f^\prime (0) = \frac{6}{100} \amp \amp f^\prime (4) = \frac{-2}{4} = \frac{-1}{2} \\ \amp \text{Positive} \amp \amp \text{Negative} \\ \amp f \text{ is increasing.} \amp \amp f \text{ is decreasing.} \end{align*}

Since the function change from increasing to decreasing at the critical point, we must have a minimum at \(x = 3\text{.}\)

Subsection 11.4.2 Optimization

Activity 11.4.5.

The function

\begin{equation*} h(t) = - 3t^2 + 9 t + 7 \end{equation*}

measures height as a function of time. Find the maximum height.

Solution

First we take the derivative.

\begin{equation*} \frac{dh}{dt} = -6t + 9 \end{equation*}

Next we set the derivative to zero and solve.

\begin{align*} -6t + 9 \amp = 0 \\ t \amp = \frac{3}{2} \end{align*}

The domain of this function is all real numbers, but we can assume \(t \geq \) for the model. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(0 \frac{3}{2} \right) \amp \amp \left(\frac{3}{2}, \infty \right)\\ \amp f^\prime (1) = 3 \amp \amp f^\prime (2) = -3 \\ \amp \text{Positive} \amp \amp \text{Negative} \\ \amp h \text{ is increasing.} \amp \amp h \text{ is decreasing.} \end{align*}

Since the function change from increasing to decreasing at the critical point, we must have a maximum at \(t = \frac{3}{2}\text{.}\) The value at \(t = \frac{3}{2}\) is \(-3 \left( \frac{3}{2} \right)^2 + 9 \frac{3}{2} + 7 = \frac{-27}{4} + \frac{27}{2} + 7 = \frac{27}{4} + \frac{28}{4} = \frac{55}{4}\text{.}\) The maximum height is \(\frac{55}{4}\) meters.

Activity 11.4.6.

A chemical reaction is controlled by a catalyst. The function

\begin{equation*} T(c) = 14\sqrt{7 - 4x + x^2} \end{equation*}

measure a bounding temperature of the reaction depending on the concentration of the catalyst. Find the minimum temperature bound.

Solution

First we take the derivative, using the chain rule (I haven't shown the steps).

\begin{equation*} \frac{dT}{dc} = \frac{14 (2x-4)}{2\sqrt{7 - 4x + x^2}} = \frac{14x - 28}{\sqrt{7 - 4x + x^2}} \end{equation*}

Next we set the derivative to zero and solve. We only need to consider the numerator.

\begin{align*} 14x - 28 \amp = 0 \\ x \amp = 2 \end{align*}

The domain of this function is all real numbers, since the quadratic in the square root is always positive. However, concentration cannot be negative, so in the model we will assume a domain of positive numbers. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(0, 2 \right) \amp \amp \left( 2, \infty \right)\\ \amp f^\prime (1) = \frac{14-28}{\sqrt{7 - 4 + 1}} = \frac{-14}{\sqrt{3}} \amp \amp f^\prime (3) = \frac{42 - 28}{\sqrt{7 - 12 + 9}} = \frac{14}{2} = 7 \\ \amp \text{Negative} \amp \amp \text{Positive} \\ \amp T \text{ is decreasing.} \amp \amp T \text{ is increasing.} \end{align*}

Since the function change from decreasing to increasing at the critical point, we must have a minimum at \(c = 2\text{.}\) The value of the function at \(c=2\) is \(T(2) = 14 \sqrt{7 - 8 + 4} = 14 \sqrt{3} \doteq 24.25\text{.}\) The minimum bounding temperature if 24.25 degree celcius.

Activity 11.4.7.

The function

\begin{equation*} T(m) = n(4m - 15\ln(m-25)) \end{equation*}

bounds the sort time of a sorting algorithm which runs on \(m\) parallel processors (for \(m\) at least 25). \(n\) is the length of the list. How many processors give the shortest sort time?

Solution

We replace \(n\) with 10000 in the function.

\begin{equation*} T(m) = 10000(4m - 15 \ln (m-25) \end{equation*}

Then we take the derivative.

\begin{equation*} \frac{dT}{dm} = 10000(4 - 15 \frac{1}{m-25}) \end{equation*}

Next we set the derivative to zero and solve.

\begin{align*} 4 - 15 \frac{1}{m-25} \amp = 0 \\ m - 25 \amp = \frac{15}{4} \\ m \amp = \frac{15}{4} + 25 = \frac{115}{4} \end{align*}

The domain of this function is all numbers above 25. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(25, \frac{115}{4} \right) \amp \amp \left( \frac{115}{4}, \infty \right)\\ \amp f^\prime (26) = 10000 (4 - 15 \frac{1}{1}) = -1100000 \amp \amp f^\prime (35) = 10000 (4 - 15 \frac{1}{10} = 25000\\ \amp \text{Negative} \amp \amp \text{Positive} \\ \amp f \text{ is decreasing.} \amp \amp f \text{ is increasing.} \end{align*}

Since the function change from decreasing to increasing at the critical point, we must have a minimum at \(m = \frac{115}{4} \text{.}\) The closest whole number to this is 29, so the ideal choice is 29 processors.

Activity 11.4.8.

The function

\begin{equation*} E(h) = \frac{-5}{3000}h^3 + \frac{1}{25}h^2 + 5 \end{equation*}

measure the fuel efficiency of an aircraft at height \(h\) (in kilometers). Find the height with the maximum fuel efficiency.

Solution

First we take the derivative.

\begin{equation*} \frac{dE}{dh} = -\frac{1}{200}h^2 + \frac{2}{25}h \end{equation*}

Next we set the derivative to zero and solve.

\begin{align*} \frac{-1}{200}h^2 + \frac{2}{25}h \amp = 0 \\ h \amp = 0 \text{ or } h = 16 \end{align*}

The domain of this function is all real numbers, but the model only makes sense for positive numbers. We ignore the critical point at 0, since we interested in positive heights. We split up this domain using the critical point to get the intervals. On each interval, we test the sign of the derivative.

\begin{align*} \amp \left(0, 16 \right) \amp \amp \left(16, \infty \right)\\ \amp f^\prime (1) = \frac{-1}{200} + \frac{2}{25} = \frac{15}{200} \amp \amp f^\prime (20) = \frac{-400}{200} + \frac{40}{25} = -2 + \frac{8}{5} = \frac{-3}{5} \\ \amp \text{Positive} \amp \amp \text{Negative} \\ \amp f \text{ is increasing.} \amp \amp f \text{ is decreasing.} \end{align*}

Subsection 11.4.3 Marginal Analysis

Activity 11.4.9.

Consider the following cost and benefit functions.

\begin{align*} C(x) \amp = \frac{1}{4}x^2 \\ B(x) \amp = 4 \sqrt{x} \end{align*}

Determine the greatest possible net benefit. Also find the break-even point with the highest gross benefit. Given an interpretation of the cost and benefit functions, with comment on their reasonability.

Solution

We have a possible maximum net benefit when the marginal cost and marginal benefit are equal. We calculate those two and set them equal to each other.

\begin{align*} \frac{d}{dx} C(x) \amp = \frac{1}{2} x \\ \frac{d}{dx} B(x) \amp = \frac{4}{2\sqrt{x}} = \frac{2}{\sqrt{x}}\\ \frac{1}{2} x \amp = \frac{2}{\sqrt{x}}\\ x^\frac{3}{2} \amp = 4\\ x \amp = 4^\frac{2}{3} = \sqrt[3]{16} \doteq 2.52 \end{align*}

Then we look at intervals. Below this critical value, say at \(x=1\text{,}\) the marginal benefit is 2 and the marginal cost is \(\frac{1}{2}\text{.}\) Here the net benefit is increasing, since the marginal benefit exceeds the marginal cost. Above the critical value, say at \(x=4\text{,}\) the marginal cost is 2 and the marginal benefit is 1. The marginal cost exceeds the marginal benefit, so the net benefit is decreasing. We conclude that we have a maximum net benefit at approximatly \(x = 2.52\)2 and the marginal benefit is 1. The marginal cost exceeds the marginal benefit, so the net benefit is decreasing. We conclude that we have a maximum net benefit at approximatly \(x = 2.52\)

For the break even point, we set the two functions each to each other and solve.

\begin{align*} C(x) \amp = B(x) \\ \frac{1}{4} x^2 \amp = 4 \sqrt{x} \\ x^\frac{3}{2} \amp = 16\\ x \amp = \sqrt[3]{256} \doteq 6.34 \end{align*}

This is only one positive break even point, at approximately \(x = 6.34\text{,}\) so that is by default the largest. FigureÂ 11.4.1 shows the behaviour of this marginal analysis model.

These cost and benefit functions seems at least somewhat reasonable. The benefit function curves down, representing a decreasing benefit with growing production. That reasonable if the demand for a product eventually dries up. The cost function curves up, showing that producing more eventually ends up costing a great deal per unit. This might not be correct if economies of scale bring down the cost, but it might make sense if the skills or materials needed become more difficult to acquire.

Activity 11.4.11.

Consider the following cost and benefit functions.

\begin{align*} C(x) \amp = \frac{1}{9} x^3\\ B(x) \amp = \left\{ \begin{matrix} 4x \amp x \leq 2 \\ 8 \amp x > 2 \end{matrix} \right. \end{align*}
Solution

We have a possile maximum net benefit when the marginal cost and marginal benefit are equal. We calculate those two and set them equal to each other. We need to work with two pieces, one for \(x \leq 2\) and another for \(x \gt 2\text{.}\) We will start with \(x \leq 2\text{.}\)

\begin{align*} \frac{d}{dx} C(x) \amp = \frac{x^2}{3} \\ \frac{d}{dx} B(x) \amp = 4\\ \frac{x^2}{3} \amp = 4 \\ x^2 \amp = 12 \\ x \amp \dot 3.46 \end{align*}

This solution is not in the range \(x \leq 2\text{,}\) so there are no critical points on this part of the domain. \begin{align*} \frac{d}{dx} C(x) \amp = \frac{x^2}{3}\\ \frac{d}{dx} B(x) \amp = 0 \end{align*}

These can never be equal and the marginal cost is always larger than the marginal benefit. We don't have a critical point and the net benefit is always decreasing.

Even though we don't have a critical point, we have an understandable situation. Before \(x=2\text{,}\) the net benefit is increasing and after \(x=2\text{,}\) the net benefit is decreasing. Therefore, we have the maximum net benefit at the cross-over point, when \(x=2\text{.}\)

This point isn't in this section of the domain, so it is not a break even point. We also have \(x=0\) that satisfies this equality, but that isn't an interesting break even point. Now let's take \(x \gt 2\text{.}\)

We have a break even point. It is the only (non-zero) break even point and the benefit is always positive, so it must be the largest break even point. FigureÂ 11.4.3 shows the behaviour of this marginal analysis model.
