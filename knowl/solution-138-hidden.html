If we were to differentiate this, it would be a chain rule derivative. The outside function would remain unchanged, but the inside derivative is \(\frac{d}{dx} 4x = 4\text{,}\) so we would have to multiply by 4. Going backwards, then, we should do the opposite and divide by 4.

\begin{equation*} \int e^{4x} dx = \frac{1}{4} e^{4x} + C \end{equation*}
