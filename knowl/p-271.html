The numerator and denominator both approach zero. This is an indeterminate form of type \(\frac{0}{0}\text{.}\) We need to do some manipulations to deal with it. Both terms are quadratic, so we will try to factor them to simplify the limit.

\begin{align*} \lim_{x \rightarrow 2} \frac{x^2 - 5x + 6}{x^2 - 7x + 10} \amp = \lim_{x \rightarrow 2} \frac{(x-2)(x-3)}{(x-2)(x-5)} \\ \amp = \lim_{x \rightarrow 2} \frac{x-3}{x-5} \end{align*}

After cancelling the \((x-2)\) term, this is now a limit that can be directly evaluated.

\begin{equation*} \lim_{x \rightarrow 2} \frac{x-3}{x-5} = \frac{2-3}{2-5} = \frac{-1}{-3} = \frac{1}{3} \end{equation*}
