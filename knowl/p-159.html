For the composition \(f \circ g\text{,}\) \(f\) is the outside function and \(g\) is the inside function. We use the function \(g\) to replace the variable in the function \(f\text{.}\)

\begin{equation*} f \circ g(x) = e^{(\sqrt{x^3-5})^2-1} = e^{x^3-5-1} = e^{x^3-6} \end{equation*}

The resulting function have no obvious domain restrictons. However, since we have to do the inside function \(g\) first, we need to respect the domain restrictions of \(g\text{.}\) The argument in the square root needs to be positive, we we need \(x^3-5 \geq 0\text{.}\) This inequality simplifies into \(x^3 \geq 5\) and further to \(x \geq \sqrt[3]{5}\text{.}\) Therefore, the domain of the composition is all real number \(x \geq \sqrt[3]{5}\text{.}\)
