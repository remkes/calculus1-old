We can also use the chain rule with a neat little exponential trick to differentiate \(f(x) = x^x\text{.}\) (In this calculation, we use \(\frac{d}{dx} \ln x = \frac{1}{x}\text{,}\) which we will prove in the next section.)

\begin{align*} \frac{d}{dx} x^x \amp = \frac{d}{dx} e^{x\ln x} = e^{x\ln x} \frac{d}{dx} x\ln x = x^x (\ln x \frac{d}{dx} x + x \frac{d}{dx} \ln x) = x^x(\ln x + 1) \end{align*}
