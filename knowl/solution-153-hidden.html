We follow the process from the notes and the videos for solving the population equation. We start with dividing by \(p(t)\text{.}\)
\begin{equation*} \frac{1}{p(t)} \frac{dp}{dt} = -0.14 \end{equation*}
Then we integrate both sides of the equation (the independent variable here is \(t\text{.}\))
\begin{equation*} \int \frac{1}{p(t)} \frac{dp}{dt} dt = \int (-0.14) dt \end{equation*}
The right side is just the integral of a constant. For the left side, we use a substitution \(u = p(t)\text{,}\) so that \(du = \frac{dp}{dt} dt \text{.}\) With that substitution, we can replace all the pieces on the right side.
\begin{equation*} \int \frac{1}{u} du = (-0.14t) + C \end{equation*}
Then we solve the left side integral. I won't use absolute value bars in the integral, since \(u = p(t)\) means that \(u\) is always positive (since population is always positive).
\begin{equation*} \ln u = (-0.14t) + C \end{equation*}
Then we can solve for u.
\begin{equation*} u = e^{(-0.14t) + C} = e^C e^{-0.14t} \end{equation*}
Now we replace u.
\begin{equation*} p(t) = e^C e^{-0.14t} \end{equation*}
Finally, we use the initial value p(0) = 150000
\begin{equation*} p(0) = e^C e^0 = e^C = 1500000 \end{equation*}
This lets us write the final solution.
\begin{equation*} p(t) = 150000 e^{-0.14t} \end{equation*}
This is exactly what we expect: population that starts at 150000 with a growth rate coefficient of \(-0.14\text{.}\)
