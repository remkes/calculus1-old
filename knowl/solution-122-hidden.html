We need a pattern. This is an arithmetic sequence, where we add 3 in each step. We start with a value of 1. Using this, we can start at \(i=0\) and describe this pattern as \(1 + 3i\text{.}\) The value of \(i\) that gives 19 is \(i=6\text{,}\) so our sum will end at 6.

\begin{equation*} 1 + 4 + 7 + 10 + 13 + 16 + 19 = \sum_{i=0}^6 (1+3i) \end{equation*}
