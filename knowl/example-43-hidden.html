A very classic (if somewhat contrived) example of an optmization problem is maximing the area of a rectangle with fixed perimeter. Let's say that \(P\) is the fixed perimeter and the rectangle has height \(h\) and length \(l\text{,}\) as in FigureÂ 11.2.2.

We want to maximize area, so we will eventually be differentiating an area function. However, the area function is \(A = hl\text{,}\) which has two variables. We need to use the perimeter restriction to eliminate one of the variables. We know \(P = 2h + 2l\) so \(h = \frac{P}{2} - l\text{.}\) If we substitute for \(h\) in the area function, we get a single variable area function \(A(l)\text{.}\)

\begin{equation*} A(l) = l \left( \frac{P}{2} - l \right) \end{equation*}

Then we can optimize. The derivative is \(A^\prime(l) = \frac{P}{2} - 2l\text{.}\) This vanishes when \(l = \frac{P}{4}\text{.}\) We can test that the critical point is a maximum. Unsurprisingly, the result shows that a square (where both \(l\) and \(h\) are exactly one quarter of the perimeter) maximizes area.
