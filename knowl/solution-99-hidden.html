This is a composition, so we use the chain rule. I'll label the pieces: we'll call the outside function \(f(u) = \cos (u) \text{,}\) using the temporary variable \(u\text{.}\) The inside function is \(g(x) = e^{-4x} \text{.}\) Then we proceed with the chain rule.

\begin{equation*} \frac{d}{dx} \cos \left( e^{-4x} \right) = \frac{d}{du} \cos u \Big|_{u = e^{-4x}} \frac{d}{dx} e^{-4x} \end{equation*}

The derivative of cosine is known. However, this second derivative is still a composition, so we need to use chain rule again. To avoid confusion of variables, I'll use a new temporary variable \(v\text{.}\) The outside is \(f(v) = e^v\) and the inside is \(g(x) = -4x \text{.}\) We apply the chain rule again using this decomposition.

\begin{equation*} = -\sin u \Big|_{u = e^{-4x}} \frac{d}{dv} e^v \Big|_{v = -4x} \frac{d}{dx} (-4x) \end{equation*}

Then we proceed with the known derivatives, replacing \(u\) and \(v\) when we have finished the derivatives.

\begin{align*} \amp = -\sin \left( e^{-4x} \right) e^v \Big|_{v = -4x} (-4) \\ \amp = -\sin \left( e^{-4x} \right) e^{-4x} (-4) = -4 e^{-4x} \sin \left( e^{-4x} \right) \end{align*}

Note that we could have chosen a different decomposition. I made the choice I made to make the outside function as simple as possible. When you have a complication composition of several functions, work from the outside in. Make the outside function easily differentiable, and apply more rules to the inside function if necessary.
