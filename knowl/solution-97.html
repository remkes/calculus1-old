Solution 7.4.3.1.

This is a composition, so we use the chain rule. I'll label the pieces: we'll call the outside function \(f(u) = \frac{1}{u} = u^{-1}\text{,}\) using the temporary variable \(u\text{.}\) The inside function is \(g(x) = 1 + e^x \text{.}\) Then we proceed with the chain rule.

\begin{equation*} \frac{d}{dx} \frac{1}{1+e^x} = \frac{d}{du} u^{-1} \Big|_{u = 1 + e^x} \frac{d}{dx} (1+e^x) \end{equation*}

The first derivative is a power rule derivative. Linearity and the known derivative of the exponential function give us the second derivative.

\begin{equation*} = -u^{-2} \Big|_{u = 1 + e^x} (e^x) \end{equation*}

Then we replace \(u\text{.}\)

\begin{equation*} = \frac{-1}{(1+e^x)^2} (e^x) = \frac{-e^x}{(1+e^x)^2} \end{equation*}
