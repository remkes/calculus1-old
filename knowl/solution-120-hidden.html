Starting with 0, we replace \(i\) with all the numbers up to 15 and then add all the resulting numbers together.

\begin{align*} \amp \sum_{i=0}^{15} 2^i = 2^0 + 2^1 + 2^2 + 2^3 + 2^4 + 2^5 + 2^6 + 2^7 + 2^8 \\ \amp + 2^9 + 2^{10} + 2^{11} + 2^{12} + 2^{13} + 2^{14} + 2^{15} \\ \amp = 1 + 2 + 4 + 8 + 16 + 31 + 64 + 128 + 256 + 512 + 1024\\ \amp + 2048 + 4096 + 8192 + 16384 + 32768 \end{align*}
