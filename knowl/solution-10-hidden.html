Solving for intesection is algebraically the same as solving the system of two equations. In these two equations, the \(y\) terms are already isolated, so we can simply equate these two terms and solve for \(x\text{.}\)

\begin{align*} -2x - 5 \amp = 9x + 1\\ -2x - 9x \amp = 5 + 1\\ -11x \amp = 6 \implies x = \frac{-6}{11} \end{align*}

Then we can insert this \(x\) value into either equation to get the \(y\) value of the intersection point. (Either equation will give the same \(y\) value; if they don't, we've made an error somewhere).

\begin{equation*} y = 9x + 1 = 9 \frac{-6}{11} + 1 = \frac{-54}{11} + \frac{11}{11} = \frac{-43}{11} \end{equation*}

We conclude that the intersection point is \(\left( \frac{-6}{11}, \frac{-43}{11} \right)\)
