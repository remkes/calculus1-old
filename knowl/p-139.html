In order for a function to be invertible, each output needs to go back to a unique input. This restrictions implies that no two inputs can be sent to the same output. This property is called injectivity, but for our purposes, it is enough that a function is monotonic. A function which always increases or decreases can only have one input sent to any particular output; therefore, it is invertible. When we want to invert a function, we must make sure that it is monotonic.
