There is a three step procedure to calculating limits.

  • First, try to evaluate the function at the limit point. For reasonable funciton (specifically, continuous funcitons, which will be defined later), if the function can be evaluated directly, the function value will be your limit value.

  • If the first method fails, try to work out logically what the limit should do.

  • If both methods fail, the limit is called an indeterminante form. In this case, we need to use various algebraic tricks (factoring, expanding, multiplying by conjugates, trig identities) and rules for manipulating limits to change the limit into a more approachable form.
