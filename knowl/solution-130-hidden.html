There is a general form for \(x_k^*\) in notes.

\begin{equation*} x_k^* = a + k \frac{b-a}{n} \end{equation*}

Now we put in the values for this particular integral.

\begin{equation*} x_k^* = -1 + k \frac{1-(-1)}{n} = -1 + \frac{2k}{n} \end{equation*}

Then we can return to the form from the previous section and replace \(x_k^*\) with this new expression. I'll proceed to simplify the polynomial after I've substituted\(x_k^*\text{.}\)

\begin{align*} \int_{-1}^1 (x^2+2+2) dx \amp = \lim_{n \rightarrow \infty} \sum_{k=0}^n \left( \left( -1 + \frac{2k}{n} \right)^2 + 2 \left( -1 + \frac{2k}{n} \right) + 2 \right) \left( \frac{2}{n} \right) \\ \amp = \lim_{n \rightarrow \infty} \sum_{k=0}^n \left( 1 - \frac{4k}{n} + \frac{4k^2}{n^2} + -2 - \frac{4k}{n} + 2 \right) \left( \frac{2}{n} \right) \\ \amp = \lim_{n \rightarrow \infty} \sum_{k=0}^n \left( 1 + \frac{4k^2}{n^2} \right) \left( \frac{2}{n} \right) \\ \amp = \lim_{n \rightarrow \infty} \sum_{k=0}^n \left( \frac{2}{n} + \frac{8k^2}{n^3} \right) \end{align*}
