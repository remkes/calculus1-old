Example 7.2.1.

There are other interesting properties of the higher derivatvies of trigonometric function.

\begin{align*} f(x) \amp = \sin x\\ f^\prime(x) \amp = \cos x\\ f^{\prime\prime}(x) \amp = -\sin x\\ f^{\prime\prime\prime}(x) \amp = -\cos x\\ f^{(4)}(x) \amp = \sin x \end{align*}

If we take four derivatives, we get back to the original function. The same is true for \(\cos x\text{,}\) but those two functions are unique among functions for this property.
