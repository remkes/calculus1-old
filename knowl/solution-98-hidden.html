This is a composition, so we use the chain rule. I'll label the pieces: we'll call the outside function \(f(u) = \sqrt[4]{u} = u^\frac{1}{4} \text{,}\) using the temporary variable \(u\text{.}\) The inside function is \(g(x) = x^2 + 4x + 2\text{.}\) Then we proceed with the chain rule.

\begin{equation*} \frac{d}{dx} \sqrt[4]{x^2+4x+2} = \frac{d}{du} u^\frac{1}{4} \Big|_{u = x^2 + 4x + 2} \frac{d}{dx} (x^2 + 4x + 2) \end{equation*}

We use power rule for the first derivative (which is why we wrote the fourth root as a fractional exponent). The polynomial derivative is solved by linearity and the power rule.

\begin{equation*} = \frac{1}{4} u^\frac{-3}{4} \Big|_{u = x^2 + 4x + 2} (2x + 4) \end{equation*}

Then we just have to do the replacement. I'll also write the fractional exponent as a root again.

\begin{equation*} = \frac{1}{4\sqrt[4]{u^3}} (2x + 4) = \frac{2x+4}{4 \sqrt[4]{x^2+4x+2)^3}} \end{equation*}
