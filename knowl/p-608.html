We need a pattern. This is a geometric progression, where we multiply by 2 in each ste. We start with a value of 3. Using this, we can start at \(i=0\) and describe this pattern as \(3(2^i)\text{.}\) The value of \(i\) that gives 96 is \(i = 5\text{,}\) so our sum will end at 6.

\begin{equation*} 3 + 6 + 12 + 24 + 48 + 96 = \sum_{i=0}^5 3(2^i) \end{equation*}
