The behaviour of an object on a (perfect, frictionless) spring is determined by this differential equation. What is that behaviour? We simply have to figure out which function \(p(t)\) matches this differential equation. In this case, there are two solutions:

\begin{align*} p(t) \amp = \sin \left( \sqrt{\frac{m}{k}} t \right) \amp \text{or} \amp \amp p(t) \amp= \cos \left( \sqrt{\frac{m}{k}} t \right) \end{align*}
