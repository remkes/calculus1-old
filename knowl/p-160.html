For the composition \(g \circ f\text{,}\) \(g\) is the outside function and \(f\) is the inside function. We use the function \(f\) to replace the variable in the function \(g\text{.}\)

\begin{equation*} g \circ f(x) = \sqrt{(e^{x^2-1})^3 - 5} = \sqrt{e^{3(x^2-1)} - 5} \end{equation*}

We've simplified this one step using the laws of exponents. Even though this is still a complicated expression, there aren't any more reasonable simplification. The domain restrictions are also complicated here. The inside function \(f\) has no restrictions, so we only have to worry about the final form; unfortunately, that final form isn't easy to pull apart. The square root is the only restrict, so the term inside the square root must be non-negative. Let's work with that inequality a bit. (We're going to apply a logarithm to get ride of the exponential; this respect the inequality, since the logarithm is an increasing function. Any increasing function preserves inequalities and an decreasing function reverses them.)

\begin{align*} e^{3(x^2-1)} - 5 \amp \geq 0 \\ e^{3(x^2-1)} \amp \geq 5 \\ \ln (e^{3(x^2-1)}) \amp \geq \ln 5 \\ 3(x^2-1) \amp \geq \ln 5 \\ x^2-1 \amp \geq \frac{1}{3} \ln 5 \\ x^2 \amp \geq \frac{1}{3} \ln 5 + 1 \\ x \amp \geq \sqrt{\frac{1}{3} \ln 5 + 1} \end{align*}

After the manipulation of this inequality, we do get an understandable domain restriction. This composition is define for all real numbers \(x \geq \sqrt{\frac{1}{3} \ln 5 + 1}\text{.}\)
