Solution 8.4.18.1.

We look to our tables of derivatives. If we differentiate \(\ln x\text{,}\) we get \(\frac{1}{x}\text{,}\) so we go backwards to that original function.

\begin{equation*} \int \frac{1}{x} dx = \ln x + C \end{equation*}

If we look at the tables of integrals, we can actually do a bit better than this. The right side of the above equation is only defined for \(x > 0\text{,}\) but the left side is defined for all \(x \neq 0\text{.}\) There is an antiderivative that matches the full domain.

\begin{equation*} \int \frac{1}{x} dx = \ln |x| + C \end{equation*}
