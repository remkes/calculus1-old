Consider this function: \(f(x) = xe^{-x^2}\text{.}\)

  1. The domain is all of \(\RR\text{.}\)

  2. The range is uncertain at first glance.

  3. The function is continuous.

  4. \(f(0) = 0\) and that is the only intercept.

  5. The function is odd.

  6. There are no undefined points, so there are no vertical asymptotoves. The limits as \(x \rightarrow \pm \infty\) are both \(0\text{,}\) so there are horizontal asymptotes \(y=0\) in both the positive and negative directions.

  7. The first derivative is \(f^\prime = e^{-x^2} -2x^2e^{-x^2}\text{.}\) This vanishes when \(x= \pm \frac{1}{\sqrt{2}}\text{.}\)

    \begin{align*} \amp (-\infty, -\sqrt{2} ) \amp \amp (-\sqrt{2}, \sqrt{2}) \amp \amp (\sqrt{2}, \infty) \\ \amp f^\prime (-2) = -7e^{-4} \amp \amp f^\prime(0) = 1 \amp \amp f^\prime(2) = -7e^{-4} \\ \amp f^\prime(x) \lt 0 \amp \amp f^\prime(x) \gt 0 \amp \amp f^\prime(x) \lt 0 \\ \amp f(x) \text{ is decreasing} \amp \amp f(x) \text{ is increasing} \amp \amp f(x) \text{ is decreasing} \end{align*}

    The point \(\left( \frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2e}} \right)\) is a local minimum and the point \(\left( \frac{-1}{\sqrt{2}}, \frac{-1}{\sqrt{2e}} \right)\) is a local maximum. This calculation also gives the range, since these extrema are also global extrema. The range is \(\left[ -\frac{1}{\sqrt{2e}}, \frac{1}{\sqrt{2e}} \right]\text{.}\)

  8. The second derivative is \(f^{\prime\prime} = -6xe^{-x^2} + 4x^3e^{-x^2}\text{.}\) This vanishes when \(x = \pm \sqrt{\frac{3}{2}}\text{.}\) which gives inflection points \(\left( \sqrt{\frac{3}{2}}, \sqrt{\frac{3}{2e^3}} \right)\) and \(\left( -\sqrt{\frac{3}{2}}, \sqrt{\frac{3}{2e^3}} \right)\text{.}\)

    \begin{align*} \amp \left( -\infty, -\sqrt{\frac{3}{2}} \right) \amp \amp \left( -\sqrt{\frac{3}{2}}, \sqrt{\frac{3}{2}} \right) \amp \amp \left( \sqrt{\frac{3}{2}}, \infty \right) \\ \amp f^{\prime \prime}(-2) = 44e^{-4} \amp \amp f^{\prime \prime}(0) = -2 \amp \amp f^{\prime \prime}(2) = 44e^{-4}\\ \amp f^{\prime \prime}(x) \gt 0 \amp \amp f^{\prime \prime}(x) \lt 0 \amp \amp f^{\prime \prime}(x) \gt 0 \\ \amp f \text{ is concave up} \amp \amp f \text{ is concave down} \amp \amp f \text{ is concave up} \end{align*}
