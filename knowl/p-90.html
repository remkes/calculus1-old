In order to define functions, we need to define sets. A set is any collection of things; the things in a set are called its elements While most of our sets will be sets of numbers, apriori a set can have any kind of elements. Particularly useful sets for this course are the standard number sets (\(\NN\text{,}\) the natural numbers; \(\ZZ\text{,}\) the integers; \(\QQ\text{,}\) the rational number, \(\RR\text{,}\) the real numbers), open inverals \((a,b)\) and closed interval \([a,b]\text{.}\)
