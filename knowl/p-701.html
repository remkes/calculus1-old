The plurals of these terms are local minima and local maxima; if we want to refer to both, we will simply say local extrema. We can also define global extrema.

  • The point \((c,f(c))\) is a global maximum if \(f(c) > f(x)\) for all \(x\) in the domain of the function.

  • The point is a global minimum if \(f(c) \lt f(x)\) for all \(x\) in the domain.
