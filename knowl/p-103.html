Linear functions have the form \(f(x) = ax + b\) for real numbers \(a\) and \(b\text{.}\) Their graphs which are straight lines, hence the name ‘linear’. Linear functions includes constant functions, since we allow \(a=0\text{.}\) All the tools from analytic geometry for understanding lines are useful for understanding linear functions. Linear functions have no domain restrictions.
