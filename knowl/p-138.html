The last important operation on functions is inversion. Inversion attempts to undo what a function has accomplished, to go backwards and return the output to the original input. If \(f(x)\) is a function, we write \(f^{-1}(x)\) for its inverse. This notation does not mean the reciprocal function \(\frac{1}{f(x)}\text{.}\)
