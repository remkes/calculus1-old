We look at the first derivative: \(f^\prime(x) = \ln x + 1\text{,}\) so \(f^\prime(x) = 0\) implies \(x = \frac{1}{e}\text{.}\) The point \(\left( \frac{1}{e}, \frac{-1}{e} \right)\) is a potential extrema.

\begin{align*} \amp \left( 0, \frac{1}{e} \right) \amp \amp \left( \frac{1}{e}, \infty \right) \\ \amp f^\prime \left( \frac{1}{e^2} \right) = -1 \amp \amp f^\prime (e) = 2 \\ \amp f^\prime(x) \lt 0 \amp \amp f^\prime(x) \gt 0 \\ \amp f(x) \text{ is decreasing} \amp \amp f(x) \text{ is increasing} \end{align*}

The point \(\left( \frac{1}{e}, \frac{-1}{e} \right)\) is a local minimum.
