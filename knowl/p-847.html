Range again: We know the function diverges to infinity as \(x \righarrow -\infty\text{,}\) so there is no upper bound. However, the minimum is a lower bound, since the function has a horizontal asymptote in the positive direction. The minimum happens at \(x = \frac{-1-\sqrt{37}}{2}\text{.}\) Evaluating the function at this point is awkward, but I asked a computer for an approximate value: \(f \left( \frac{-1-\sqrt{37}}{2} \right) \doteq -140.915\text{.}\) Therefore, the range (in approximate values) if \((-140.914, \infty)\)
