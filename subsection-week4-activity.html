Section 4.5 Week 4 Activity

Subsection 4.5.1 Limits at Finite Values

Activity 4.5.1.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 3} 4x^2 -5 \end{equation*}
Solution

There are no domain restrictions here; this limit can be directly evaluated.

\begin{equation*} \lim_{x \rightarrow 3} 4x^2 -5 = 4(3)^2 - 5 = 31 \end{equation*}
Activity 4.5.2.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 3} \frac{1}{x-3} \end{equation*}
Solution

We cannot directly evaluation, since that would lead to division by zero. However, we can logically analyze this limit. The numerator is fixed and the denominator is getting very small. Division by smaller and smaller numbers leads to larger and larger numbers. We conclude the limit must be infinite. The denominator can be negative or positive, so we write

\begin{equation*} \lim_{x \rightarrow 3} \frac{1}{x-3} = \pm \infty \end{equation*}
Activity 4.5.3.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 2} \frac{x^2 - 5x + 6}{x^2 - 7x + 10} \end{equation*}
Solution

The numerator and denominator both approach zero. This is an indeterminate form of type \(\frac{0}{0}\text{.}\) We need to do some manipulations to deal with it. Both terms are quadratic, so we will try to factor them to simplify the limit.

\begin{align*} \lim_{x \rightarrow 2} \frac{x^2 - 5x + 6}{x^2 - 7x + 10} \amp = \lim_{x \rightarrow 2} \frac{(x-2)(x-3)}{(x-2)(x-5)} \\ \amp = \lim_{x \rightarrow 2} \frac{x-3}{x-5} \end{align*}

After cancelling the \((x-2)\) term, this is now a limit that can be directly evaluated.

\begin{equation*} \lim_{x \rightarrow 2} \frac{x-3}{x-5} = \frac{2-3}{2-5} = \frac{-1}{-3} = \frac{1}{3} \end{equation*}
Activity 4.5.4.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow -3} \frac{x+3}{x^3+27} \end{equation*}
Solution

Both the numerator and denominator approach zero, so this is an indeterminate form of type \(\frac{0}{0}\text{.}\) We can't do anything to the numerator, but we can factor the denominator to try to simplify the limit.

\begin{align*} \lim_{x \rightarrow -3} \frac{x+3}{x^3+27} \amp 0 \lim_{x \rightarrow -3} \frac{(x+3)}{(x+3)(x^2-3x+9} \amp = \lim_{x \rightarrow -3} \frac{1}{x^2-3x+9} \end{align*}

This is a limit we can now evaluate.

\begin{equation*} \lim_{x \rightarrow -3} \frac{1}{x^2-3x+9} = \frac{1}{(-3)^2 - 3(-3) + 9} = \frac{1}{27} \end{equation*}
Activity 4.5.5.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 0} \frac{x^4+3x^-8x^2}{x^4 + 6x^3 + 10x^2} \end{equation*}
Solution

Both numerator and denominator approach zero, so this is an indeterminate form of type \(\frac{0}{0}\text{.}\) We can factor \(x^2\) out of both numerator and denominator to simplify the limit. After that factoring, we can direclty evaluate.

\begin{align*} \lim_{x \rightarrow 0} \frac{x^4+3x^-8x^2}{x^4 + 6x^3 + 10x^2} \amp = \lim_{x \rightarrow 0} \frac{x^2(x^2+3x-8)}{x^2(x^2+6x+10} \\ \amp = \lim_{x \rightarrow 0} \frac{x^2+3x-8}{x^2+6x+10} = \frac{-8}{10} = \frac{-4}{5} \end{align*}
Activity 4.5.6.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow \frac{1}{2}} \frac{4x^2 + 4x - 3}{8x^2 - 2x -1} \end{equation*}
Solution

Both numerator and denominator approach 0, so this is a limit of indeterminate form \(\frac{0}{0}\text{.}\) Both terms are quadratics, so we will try to factor the quadratic to simplify the limit.

\begin{align*} \lim_{x \rightarrow \frac{1}{2}} \frac{4x^2 + 4x - 3}{8x^2 - 2x -1} \amp = \lim_{x \rightarrow \frac{1}{2}} \frac{(2x-1)(2x+3)}{(2x-1)(4x+1)}\\ \amp = \lim_{x \rightarrow \frac{1}{2}} \frac{2x+3}{4x+1} \end{align*}

This is a limit we can now directly evaluate.

\begin{equation*} \lim_{x \rightarrow \frac{1}{2}} \frac{2x+3}{4x+1} = \frac{1+3}{2+1} = \frac{4}{3} \end{equation*}
Activity 4.5.7.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 0} \frac{\sin^2 x}{x} \end{equation*}
Solution

We have a limit involving \(x\) and \(\sin x\text{.}\) We want to use the known limit for the ratio of these two functions. We manipulate this limit to isolate the known limit, using the rules for limits and multiplication.

\begin{equation*} \lim_{x \rightarrow 0} \frac{\sin^2 x}{x} = \lim_{x \rightarrow 0} \frac{\sin x}{x} \lim_{x \rightarrow 0} \sin x \end{equation*}

Then the first part of this is the known limit, which evaluates to 1. The second part can be directly evaluated to 0. The product of 1 and 0 is 0, so the limit is 0.

Activity 4.5.8.

Calculate the limit

\begin{equation*} \lim_{x \rightarrow 0} x \cot x \end{equation*}
Solution

This involves trigonometry and \(x\text{,}\) so we will try to isolate the ration of \(x\) and \(\sin x\text{.}\) To do that we have to write cotagents in terms of sine and cosine.

\begin{equation*} \lim_{x \rightarrow 0} x \cot x = \lim_{x \rightarrow 0} x \frac{\cos x}{\sin x} \end{equation*}

Then we manipulate the limit to isolate the known ratio of \(x\) amd \(sin x\text{.}\)

\begin{equation*} = \lim_{x \rightarrow 0} \frac{x}{\sin x} \cos x = \lim_{x \rightarrow 0} \frac{x}{\sin x} \lim_{x \rightarrow 0} \cos x \end{equation*}

Now there are two limits, each of which evaluate to 1, so the original limit is also 1.

Subsection 4.5.2 Vertical Asymptotes

Activity 4.5.9.

Find the vertical asymptotes of the the function

\begin{equation*} f(x) = \frac{1}{x^2-4x+3} \end{equation*}
Solution

To find vertical asymptotes of a rational function, we look for the points on the edge of the domain and take the limits approaching those points. Here, domain is all real number except \(x=3\) and \(x=1\text{.}\) Those two numbers are the roots of the denominator. To determine whether we have vertical asymptotes are the undefined points, we need to take the limits approaching them. We have two limits to consider.

\begin{equation*} \lim_{x \rightarrow 1} \frac{1}{x^2-4x+3} \text{ and } \lim_{x \rightarrow 3} \frac{1}{x^2-4x+3} \end{equation*}

Both of these limits are limits where the numerator is fixed and the denominator approaches 0. In these cases, dividing by very small numbers produced very large numbers, so the limits are both \(\pm \infty\text{.}\) Since the limits are infinite, we conclucde that the lines \(x=1\) and \(x=3\) are both vertical asymptotes.

Activity 4.5.10.

Find the vertical asymptotes of the the function

\begin{equation*} f(x) = \frac{x-3}{x^2 + x - 12} \end{equation*}
Solution

To find vertical asymptotes of a rational function, we look for the points on the edge of the domain and take the limits approaching those points. Here, domain is all real number except \(x=3\) and \(x=-4\text{.}\) Those two numbers are the roots of the denominator. To determine whether we have vertical asymptotes are the undefined points, we need to take the limits approaching them. We have two limits to consider.

\begin{equation*} \lim_{x \rightarrow 3} \frac{x-3}{(x-3)(x+4)} \text{ and } \lim_{x \rightarrow -4} \frac{x-3}{(x-3)(x+4)} \end{equation*}

In the first limit, the \((x-3)\) terms cancel out and the limit can be directly evaluted to \(\frac{1}{7}\text{.}\) Since this limit if not infinite, this is not a vertical asymptote. In the second limit, the numerator is fixed and the denominator approaches 0. In this case, dividing by very small numbers produced very large numbers, so the limit is \(\pm \infty\text{.}\) Since the limit is infinite, we conclucde that the line \(x=-4\) is is a vertical asymptote.

Unlike the previous questions, this is more about familiarity with common functions and shifts in the plane. Subsection 4.5.3 Continuity

Activity 4.5.13.

Find a value for \(a\) (if there is any) which makes this piecewise function continuous.

\begin{equation*} f(x) = \left\{ \begin{matrix} x+3 \amp x > 0 \\ -2x + a \amp x \leq 0 \end {matrix} \right. \end{equation*}
Solution

To determine continuity, we need to take the limit from the left and the limit from the right approaching the cross over point. Here are the two limits

\begin{align*} \lim_{x \rightarrow 0^+} f(x) \amp = \lim_{x \rightarrow 0^+} x+3 = 3 \\ \lim_{x \rightarrow 0^-} f(x) \amp = \lim_{x \rightarrow 0^-} -2x + a = a \end{align*}

Making these equation gives the equation

\begin{equation*} 3 = a \end{equation*}

This equation is (obviously) solved when \(a=3\text{.}\)

Activity 4.5.14.

Find a value for \(a\) (if there is any) which makes this piecewise function continuous.

\begin{equation*} f(x) = \left\{ \begin{matrix} x^2 + 2x + 5 \amp x \lt -1 \\ ax^2 \amp x \geq -1 \end {matrix} \right. \end{equation*}
Solution

To determine continuity, we need to take the limit from the left and the limit from the right approaching the cross over point. Here are the two limits

\begin{align*} \lim_{x \rightarrow -1^+} f(x) \amp = \lim_{x \rightarrow -1^+} ax^2 = a\\ \lim_{x \rightarrow -1^-} f(x) \amp = \lim_{x \rightarrow -1^-} x^2 + 2x + 5 = 1 - 2 + 5 = 4 \end{align*}

Making these equation gives the equation

\begin{equation*} a = 4 \end{equation*}

This equation is (obviously) solved when \(a = 4\text{.}\)

Activity 4.5.15.

Find a value for \(a\) (if there is any) which makes this piecewise function continuous.

\begin{equation*} f(x) = \left\{ \begin{matrix} \sin (\pi x) \amp x > \frac{5}{2} \\ x^2 + ax + 2 \amp x \leq \frac{5}{2} \end {matrix} \right. \end{equation*}
Solution

To determine continuity, we need to take the limit from the left and the limit from the right approaching the cross over point. Here are the two limits

\begin{align*} \lim_{x \rightarrow \frac{5}{2}^+} f(x) \amp = \lim_{x \rightarrow \frac{5}{2}^+} \sin (\pi x) = \sin \left( \frac{5\pi}{2} \right) = 1 \\ \lim_{x \rightarrow \frac{5}{2}^-} f(x) \amp = \lim_{x \rightarrow \frac{5}{2}^-} x^2 + ax + 2 = \frac{25}{4} + \frac{5a}{2} + 2 = \frac{33 + 10a}{4} \end{align*}

Making these equation gives the equation

\begin{equation*} 1 = \frac{33 + 10a}{4} \end{equation*}

This equation is solved when \(a = \frac{-29}{10}\text{.}\)

Activity 4.5.16.

Find a value for \(a\) (if there is any) which makes this piecewise function continuous.

\begin{equation*} f(x) = \left\{ \begin{matrix} \frac{1}{x} \amp x > 0 \\ x^2 + ax + a \amp x \leq 0 \end {matrix} \right. \end{equation*}
Solution

To determine continuity, we need to take the limit from the left and the limit from the right approaching the cross over point. Here are the two limits

\begin{align*} \lim_{x \rightarrow 0^+} f(x) \amp = \lim_{x \rightarrow 0^+} \frac{1}{x} = \infty \\ \lim_{x \rightarrow 0^-} f(x) \amp = \lim_{x \rightarrow 0^-} x^2 + ax + a = a \end{align*}

We cannot equate these limits, since \(\infty\) is not a number. There is no value of \(a\) which makes this piecewise function continuous.

Subsection 4.5.4 Edge Behaviour of Models

Activity 4.5.17.

Consider a population model.

\begin{equation*} f(t) = \left\{ \begin{matrix} \frac{1}{16}e^{\frac{1}{10}x} \amp t \in [0,10] \\ \frac{39}{12}e^{\frac{1}{7}x} \amp t \in [10,20] \end{matrix} \right. \end{equation*}

Is this model continuous? If not, what happens at \(t = 20\text{?}\)

Solution

We take the limits from the left and from the right to determine continuity. The limit from the left is \(\frac{e}{16}\) from the right is \(\frac{39}{12}e^{\frac{10}{7}}\text{.}\) The approximate values of these numbers is, respectively, \(0.1699\) and \(13.5614\text{.}\) These are not the same, so the function is continuous. At \(t=10\) the population has a sudden and significant jump; through some process, a large population has suddenly been added to the existing population.

Activity 4.5.18.

Consider a model of temperature in a chemical reaction, as a function of time.

\begin{equation*} T(t) = 24 + \frac{\sqrt{(t-5)^3}}{t-5} \end{equation*}

The point \(t=5\) is the edge of the domain of this model. What happens to the temperature as we approach this edge point?

Solution

We take the limit approaching the edge point.

\begin{equation*} lim_{t \rightarrow 5} 24 + \frac{\sqrt{(t-5)^3}}{t-5} = 24 \lim_{t \rightarrow 5} + \frac{(t-5)}{(t-5)} \sqrt{t-5} = 24 + 0 = 24 \end{equation*}

Even though the function is undefined, the limit is reasonable and the temperature simply approaches 24.
