Section 5.5 Week 5 Activity

Subsection 5.5.1 Limits at Infinity

Activity 5.5.1.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{9x^2+9x-5}{4x^4-3x+102} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^2\text{.}\) The asymptotic order of the denominator is \(x^4\text{.}\) The denominator has a higher asymptotic order, so the limit is 0.

Activity 5.5.2.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{24x^5 - 85x^4 - 9x^2 + 4x}{90x^3 + 405x^2 - 817} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^5\text{.}\) The asymptotic order of the denominator is \(x^3\text{.}\) The numerator has a higher asymptotic order and the leading coefficients are both positive, the limit diverges to \(\infty\text{.}\)

Activity 5.5.3.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{132x^4 - 56 x^3 + 5x - 100}{46x^4 - 512x^3 + 5123x^2 + 4} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^4\text{.}\) The asymptotic order of the denominator is \(x^4\text{.}\) These are the same, so the limit must be the ratio of the leading coefficients.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{132x^4 - 56 x^3 + 5x - 100}{46x^4 - 512x^3 + 5123x^2 + 4} = \frac{132}{46} = \frac{66}{23} \end{equation*}
Activity 5.5.4.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{93x^2 - 40x^3 + x + 5}{x - x^2 + 82x^3} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^3\text{.}\) The asymptotic order of the denominator is \(x^3\text{.}\) The fact that the term are ordered differently doesn't change the fact that we only need the term with the highest order. Since the asymptotic orders are the same, we look at the ratio of the leading coefficients. Those coefficients are the coefficients of the highest order piece, not necessarily the first in the written order.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{93x^2 - 40x^3 + x + 5}{x - x^2 + 82x^3} = \frac{40}{82} = \frac{20}{41} \end{equation*}
Activity 5.5.5.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{8x^4 - 91542}{7x^4 + 819234} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^4\text{.}\) The asymptotic order of the denominator is \(x^4\text{.}\) The fact that the constants here are large number is irrelevant in the limit. Since the asymptotic orders are the same, the limit is the ratio of the leading coefficients.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{8x^4 - 91542}{7x^4 + 819234} = \frac{8}{7} \end{equation*}
Activity 5.5.6.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{\sqrt{x^5 - 1}}{x^2} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is determined by ignoring the 1. That leaves \(\sqrt{x^5} = x^\frac{5}{2}\text{.}\) The asymptotic order of the denominator is \(x^2\text{.}\) Since 2 is less than \(\frac{5}{2}\text{,}\) the numerator has a higher asymptotic order. Since the leading coefficients are positive and the square root produces a positive number, this limit diverges to \(\infty\text{.}\)

Activity 5.5.7.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{x}{\sqrt[3]{1-x^5}} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x\text{.}\) The asymptotic order of the denominator is found by ignoring the 1, giving \(\sqrt[3]{x^5} = x^\frac{5}{3}\text{.}\) Since \(\frac{5}{3}\) is greater than 1, the denominator has higher asymptotic order and the limit converges to 0.

Activity 5.5.8.

Evaluate this limit.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{x^2e^x + 7}{e^{2x} + 1} \end{equation*}
Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(x^2e^x\text{.}\) The asymptotic order of the denominator is \(e^{2x}\text{.}\) By the laws of exponents, \(e^{2x} = (e^x)^2 = e^x e^x \text{.}\) If we compare the two asymptotic orders and remove \(e^x\) form each, what is left in the numerator is \(x^2\) amd what is left in the denominator is \(e^x\text{.}\) In this way, we see that the denominator has higher asymptotic order, and the limit converges to 0.

Activity 5.5.9.

Evaluate this limit.

Solution

We use asymptotic analysis for limits a \(x \rightarrow \infty\text{.}\) The asymptotic order of the numerator is \(\ln x>\text{.}\) The asymptotic order of the denominator is \(\sqrt{x}\text{.}\) The denominator has higher asymptotic order, so the limit converges to 0.

\begin{equation*} \lim_{x \rightarrow \infty} \frac{5 + \ln x}{\sqrt{1+ 7x}} = \frac{1}{5} \end{equation*}

Subsection 5.5.2 Horizontal Asymptotes

Activity 5.5.12.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = \frac{7x + 4}{x^2 - 9x + 7}. \end{equation*}
Solution

The denominator has higher asymptotic order: \(x^2 > x\text{.}\) Therefore, the limit approaching \(\infty\) is 0, meaning that \(y=0\) is a horizontal asymptote in the positive direction. In the negative direction, the result is the same, so \(y=0\) is a horizontal asymptote in both directions.

Activity 5.5.13.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = \frac{8x^3 - 7x + 1}{17x^3 - 5x^2 + 12} \end{equation*}
Solution

As \(x \rightarrow \infty\text{,}\) since the asymptotic orders of numerator and denominator are the same, this approaches the ration of the leading terms, which is \(\frac{8}{17}\text{.}\) Therefore, \(y=\frac{8}{17}\) is a horizontal asymptote in the positive direction. In the negative direction, the behaviour is the same. Both of the \(x^3\) terms will be negative, but the negatives will cancel and the limits will still be \(\frac{8}{17}\text{,}\) making \(y=\frac{8}{17}\) a horizontal asymptote in both directions.

Activity 5.5.14.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = \frac{8 \ln x - 5}{6 - 12 \ln x} \end{equation*}
Solution

The asymptotic order of both numerator and denominator is \(\ln x\text{.}\) Since they are the same, the limit as \(x \rightarrow \infty\) is the limit of the leading coefficients: \(\frac{8}{12} = \frac{2}{3}\text{.}\) Therefore, \(y=\frac{2}{3}\) is a horizontal asymptote in the positive direction. There cannot e any horizontal asymptote in the negative direction, since the domain of the function is restricted to positive number by the logarithm terms.

Activity 5.5.15.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = \frac{1}{6 - \frac{4}{3-x}} \end{equation*}
Solution

As \(x \rightarrow \infty\text{,}\) the denominator of the fraction \(\frac{4}{3-x}\) approaches \(-\infty\text{.}\) Since the numerator is fixed, the fraction itself approach 0. That leaves the limit as \(\frac{1}{6-0} = \frac{1}{6}\text{.}\) Therefore, \(y = \frac{1}{6}\) is a horizontal asymptote int he positive direction. The same logic works for \(x \rightarrow -\infty\text{;}\) the only difference is that the denominator of the fraction is positive, but the conclusions are the same. Therefore, \(y = \frac{1}{6}\) is a horizontal asymptote in both directions.

Activity 5.5.16.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = 8 \arctan x - 4 \end{equation*}
Solution

Unlike the previous question, this is determined not by a calculation, but by familiarity with the arctangent function. By itself, arctangent has a horizontal asymptote of \(y = \frac{\pi}{2}\) in the positive direction and \(y = \frac{-\pi}{2}\) in the negative direction. Multiplying by 8 means that we have a horizontal asymptote of \(y = 4\pi\) in the positive direction and \(y = -4\pi\) in the negative direction. Adding 4 means that we have a horizontal asymptote of \(y = 4\pi + 4 \) in the positive direction and \(y = -4\pi + 4 \) in the negative direction.

Activity 5.5.17.

Find the horizontal asymptotes (if any) of the function

\begin{equation*} f(x) = e^{-4x^2} \end{equation*}
Solution

Since we have \(x^2\) in the function (and no other appearances of the independent variable), the behaviour in the positive and negative directions will be the same. In either direction, as \(x \rightarrow \infty\text{,}\) this is exponential decay to zero. Therefore, \(y=0\) is a horizontal asymptote in both directions.

Subsection 5.5.3 Long Term Behaviour of Models

Activity 5.5.18.

Determine the long term behaviour of a population given by the function

\begin{equation*} p(t) = 140e^{\frac{1}{40}t} \end{equation*}

Is this exponential growth, logistic growth, or something else?

Solution

This is an unbounded exponential function, so the limit is \(\infty\text{.}\) This is exponential growth, with a starting value of 140 and a growth rate coefficnet of \(\frac{1}{40}\text{.}\)

Activity 5.5.19.

Determine the long term behaviour of a population given by the function

\begin{equation*} p(t) = \frac{75e^{\frac{1}{37}t}}{34 + 420e^{\frac{1}{37}t}} \end{equation*}

Is this exponential growth, logistic growth, or something else?

Solution

Both numerator and denominator have the same asymptotic order, \(e^{\frac{1}{37} t}\text{,}\) so we look at the leading coefficients for the limit: \(\frac{75}{420} = \frac{5}{28}\text{.}\) The long term behaviour of this is stabilitzation to at the value \(\frac{5}{28}\text{.}\) This is logistic growth, since it fits the specialized logistic growth form with the same exponential functions in the numerator and denominator.

Activity 5.5.20.

Determine the long term behaviour of a population given by the function

\begin{equation*} p(t) = \frac{3t^2 - 6t + 1}{t^2 + 4} \end{equation*}

Is this exponential growth, logistic growth, or something else?

Solution

The asymptotic order of the numerator and denominator are both \(t^2\text{,}\) so the limit as \(t \rightarrow \infty\) is the ratio of the leading coefficients, which is 3. Even though this levels off, this is not logistic growth, since it doesn't conform to the specific construction of logistic growth with equivalent exponentials in both numerator and denominator.

Activity 5.5.21.

Determine the long term behaviour of a population given by the function

\begin{equation*} p(t) = 400 + \frac{t^2-1}{9t^2+4} \sin (\pi t) \end{equation*}

Is this exponential growth, logistic growth, or something else?

Solution

Looking at the sinusoidal term, the fraction before the sine function gives an amplitude. In the limit, this fraction has the same asymptotic order in the numerator and denominator, to its limit is \(\frac{1}{9}\text{,}\) the ratio of its leading coefficient. The long term behaviour of this function is osscilation around the equilibrium value of 400 with an amplitude of \(\frac{1}{9}\text{.}\) All these values are close to 400, but the function never approaches any one value; instead, the oscillations continue for ever. This is neither exponential or logistic growth.

Activity 5.5.22.

Determine the long term behaviour of a population given by the function

\begin{equation*} p(t) = \frac{7e^{\frac{2}{17}t}}{1 + 12e^{\frac{3}{27}t}} \end{equation*}

Is this exponential growth, logistic growth, or something else?

Solution

The exponential coefficient in the denominator is larger than the exponential coefficient the numerator, so the asymptotic order of the denominator is higher and the long term behaviour is decay to 0. This looks very close to logistic growth, but it isn't: the exponential coefficients are not the sam, and its doesn't level off to a non-zero carrying capacity.
